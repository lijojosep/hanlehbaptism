# Hanleh Isabel Lijo — Holy Baptism Invitation

A responsive, framework-free invitation built with HTML, CSS and vanilla JavaScript.

## Personalize

1. Replace assets/hero-placeholder.svg and assets/gallery-1.svg through gallery-4.svg with your photos (keep the same filenames, or update the paths in index.html).
2. Update the visible date, time, church, venue and contact placeholders in index.html.
3. Update EVENT_DATE and GOOGLE_MAPS_URL at the top of script.js.
4. Open index.html directly, or upload the folder to any static web host.

The RSVP and blessing form are frontend-only, as requested. Background music is generated locally with the Web Audio API and starts only after a guest taps the music control.
